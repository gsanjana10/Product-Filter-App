🛍️ Product Filter App
A responsive Product Filter App built using HTML, CSS, and JavaScript.

✨ Features
• Search products by name
• Filter products by category
• Filter products by price range
• Show only in-stock products 
• Sort products by price and name
• Responsive design

🛠️ Technologies Used
• HTML
• CSS
• JavaScript

📚 Concepts Used
• DOM Manipulation
• Event Handling
•  Array.filter() 
• Array.sort()
• Functions
c Responsive Design
